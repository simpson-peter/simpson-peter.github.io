Latin Square Solver
Visualization of a Latin square generated by my program from a set of hex codes.
Repository Link

This is a templated, efficient, implementation of a custom-sizeable Latin square that I created in C++.

The Latin square has applications to cryptography, statistics, and telecommunications, but I just made this for fun. I implemented the solver as a backtracking algorithim for maximal efficiency given the non-polynomial nature of the solution.
